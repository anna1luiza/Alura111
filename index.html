Desafio: Controle de Estoque de uma Loja

Imagine que você está desenvolvendo um sistema simples de controle de estoque para uma pequena loja. O programa deverá permitir que o usuário gerencie a entrada e saída de produtos, além de verificar se há estoque suficiente para uma operação de saída.

Instruções

O desafio deve ser resolvido em Python. O programa deve:

  • Começar com um estoque inicial predefinido.
  • Receber comandos do usuário para adicionar ou remover produtos.
  • Atualizar o estoque após cada operação.
  • Exibir uma mensagem de erro se a quantidade de saída for maior que a disponível.
  • Exibir o estoque atualizado após cada operação válida.

Entrada

  • Estoque inicial: Uma lista de produtos e suas respectivas quantidades, como um dicionário em Python. Exemplo: {'arroz': 50, 'feijao': 20, 'macarrao': 30}
  • Comandos: O programa deve ler entradas do usuário, como 'entrada arroz 10' ou 'saida feijao 5'.

Saída Esperada

  • Estoque Atualizado: Após cada operação, o programa deve imprimir o estoque atual. Exemplo: Estoque atualizado: {'arroz': 60, 'feijao': 20, 'macarrao': 30}
  • Mensagens de Erro: Se a quantidade for insuficiente, deve exibir uma mensagem como: Erro: Quantidade insuficiente de feijao. Estoque atual: 10.

Sugestão de Estrutura e Lógica

Para resolver este desafio, você pode seguir a seguinte lógica:

  • Use um dicionário para armazenar os produtos e suas quantidades.
  • Use um laço de repetição (como while) para permitir que o usuário insira comandos continuamente.
  • Use uma estrutura condicional (if/elif/else) para verificar se o comando é de entrada ou saída e para tratar a condição de estoque insuficiente.
